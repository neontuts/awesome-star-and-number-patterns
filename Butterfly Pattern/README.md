# Print a Butterfly Pattern

Write a program to print butterfly pattern. Given the value of n, print the butterfly pattern.

### For Example :-

**INPUT**
```
n = 4
```

**OUTPUT**
```
*             *
* *         * *
* * *     * * *
* * * * * * * *
* * * * * * * *
* * *     * * *
* *         * *
*             *
```
